# Pick A Meal 🍽️

Pick A Meal is a Flutter application designed to help users organize their meals, discover new recipes using AI, and solve the "what should I eat?" dilemma with a fun randomizer.

## ✨ Features

- **Meal Management**: Add, edit, and delete your favorite meals with names, descriptions, and categories.
- **Photo Support**: Attach photos to your meals using the device camera or gallery.
- **Categorization**: Organize meals into categories like Meat, Fish, Chicken, Fruit, Snacks, and more.
- **Search & Filter**: Quickly find meals by name or filter them by category.
- **Randomizer (Dice)**: Can't decide what to eat? Let the app pick a random meal for you with a fun animation.
- **AI-Powered Recipes**: Generate ingredients and cooking instructions for any meal using Google Gemini AI.
- **Theme Customization**: Support for Light Mode, Dark Mode, and System Default themes.
- **Settings**:
  - Personalize the app theme.
  - Configure your own **Gemini API Key** for unlimited recipe generation.
  - View Developer information.
- **Persistent Storage**: All your meals and settings are saved locally on your device.

## 🛠️ Built With

- **Flutter**: Cross-platform UI framework.
- **Dart**: Programming language.
- **Shared Preferences**: Local data persistence.
- **Google Generative AI (Gemini)**: For generating recipes.
- **Image Picker**: For capturing and selecting meal photos.

## 🚀 Getting Started

### Prerequisites

- Flutter SDK installed.
- An Android or iOS device/emulator.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/ahmad-bahaa//pick-a-meal.git
    cd pick-a-meal
    ```

2.  **Install dependencies**:
    ```bash
    flutter pub get
    ```

3.  **Run the app**:
    ```bash
    flutter run
    ```

## 🔑 API Configuration

To use the AI recipe generation feature, you can provide your own Gemini API key in the app's **Settings** screen. This ensures you aren't limited by shared quotas.
